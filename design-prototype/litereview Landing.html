Attention Is All You Need
Vaswani et al. · NeurIPS 2017 · arXiv:1706.03762

The dominant sequence transduction models are based on complex recurrent or convolutional neural networks that include an encoder and a decoder. We propose a new simple network architecture, the Transformer, based solely on attention mechanisms, dispensing with recurrence and convolutions entirely. Experiments on two machine translation tasks show these models to be superior in quality while being more parallelizable and requiring significantly less time to train. Our model achieves 28.4 BLEU on the WMT 2014 English-to-German translation task, improving over the existing best results. The Transformer generalizes well to other tasks by applying it successfully to English constituency parsing both with large and limited training data.

02 · 精讀 KEYPOINTS

九個欄位，
把一篇論文拆透。

不是摘要的摘要。litereview 優先抓全文 （arXiv → 上傳 PDF → 開放取用版本）， 再由 LLM 產出結構化重點——從研究問題到關鍵公式，逐欄拆解。

  • 全文抓不到時誠實標示「僅摘要」，不假裝讀過
  • 分析結果可一鍵回寫 Zotero 成為子筆記
  • 典型一篇 1–2 分鐘完成，結果永久快取在本機

04 · 攻防 DEBATE

讓模型吵一架，
你聽結論。

先由審查員給出五維 scorecard 與可辯論的爭點， 一鍵發起辯論：正反方多輪攻防、裁判評分判決， 逐字稿即時串流呈現。

  • 六個座位（找重點／比較／審查／正方／反方／裁判）可各自指派模型
  • 讓反方或裁判用不同家的模型，避免自我偏好
  • 支援 OpenAI、Gemini、Anthropic API 與任何相容端點（DeepSeek／Groq／Ollama）
LIVE · SSE STREAMING
MOTION · 來自審查爭點 自注意力的 O(n²) 複雜度，是否為長序列任務的根本瓶頸？
正方 · claude 記憶體隨序列長度平方成長是架構性事實：128k 上下文的 KV cache 成本已成部署的第一瓶頸，各種線性化嘗試至今未在品質上追平。
反方 · gemini 「瓶頸」不等於「根本」。FlashAttention 已把常數項壓低一個量級，且實務上長文任務的失效多來自訓練分布，而非注意力本身的複雜度。

HOW IT WORKS

從一個關鍵字，到一場判決。

01

搜尋或匯入

關鍵字／DOI／arXiv ID，或從 Zotero、PDF 匯入工作區。

02

找重點

抓全文、九欄位深度分析，狀態即時顯示在側欄。

03

比較

勾 2–6 篇，五維對齊成一張表 + 綜合結論。

04

審查

五維評分 scorecard、優缺點、可辯論的爭點。

05

辯論

正反攻防、裁判判決，逐字稿即時串流。

LOCAL-FIRST · OPEN SOURCE

你的研究，留在你的電腦。

沒有帳號系統、沒有雲端同步、沒有遙測。

庫

資料不出門

所有論文、分析、辯論紀錄與 API key 都存在本機 SQLite；上傳的 PDF 也只放在你的磁碟。刪掉資料夾，就真的刪乾淨了。

訂

訂閱即用，不另付費

預設模型走 claude CLI，用你既有的 Claude 訂閱登入 token——不吃 API key、不另計費。想換 OpenAI、Gemini 或本機 Ollama？設定中心隨時調度。

源

MIT 開源

整套原始碼公開在 GitHub，MIT 授權。搜尋邏輯、prompt、辯論引擎全部攤在陽光下——你可以審計它，也可以改造它。

git clone → npm install → npm run dev ，三步開始。
